## Para instalar o IONIC, primeiro precisa ter o node.js e o NPM

-----------------------------------------------------------------------

## Para instalar o IONIC

```sh
$ npm install -g ionic
```

## Para gerar a base do projeto

```sh
ionic start <NomeApp> <tipo>
$ ionic start MackFood blank
```

## Para rodar o App
```sh
$ cd MackFood
$ ionic serve
```
